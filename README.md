# lf-iac
terraform script to create vpc, public and private subnets, ecs, alb etc.

Tor run the script run following commands:
Init the terraform
```terraform init```

Plan the changes to be updated to the provider.
```terraform plan```

Apply the changes to the provider
```terraform apply```

Destroy the changes
```terraform destroy```
